Peeling back the layers
1

Open it as text, not an image

The extension lies. Open it with a text editor, or read it straight from a terminal.

cat ns7_telemetry_snapshot.jpg
2

Decode the outer layer

The whole file content is one Base64 string. Decoding it gives a readable telemetry message, not the flag yet.

base64 -d ns7_telemetry_snapshot.jpg
3

Find and decode the inner payload

Inside that message sits an ENCODED PAYLOAD section with a second Base64 string. Decode that one separately, that is where the flag actually is.

Worth remembering. Base64 is an encoding, not encryption. Anyone who finds a Base64 string can decode it instantly with no key required. Two layers of it is still zero real confidentiality.
