Hints
Hint cost: 10 XP
Real reconnaissance always covers the same checklist: HTML source, robots.txt, response headers, linked JS and CSS files. Do not rely on what the browser renders; look at raw HTTP responses. The flag is split deliberately so checking only one location is not enough.
Hint cost: 10 XP
Not every encoded string in the source is meaningful. Decode all of them. The one that contains a flag prefix is real. A string ending in == is Base64. Burp Suite Decoder tab or echo "..." | base64 -d both work.
Hint cost: 15 XP
HTTP response headers are never shown in a browser's normal view. Use Burp Suite, DevTools Network tab, or curl with -v to see them. A path returning 404 can still carry useful information in its headers.
